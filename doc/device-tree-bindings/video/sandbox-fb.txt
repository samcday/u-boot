Sandbox LCD
===========

This uses the displaymode.txt binding except that only xres and yres are
required properties. Also an additional optional property is defined:

bpp: Bits per pixel for the U-Boot display buffer. Supported values are 8, 16,
	24 and 32. If not provided, a value of 16 is used.

Example:

	lcd {
		compatible = "sandbox,lcd-sdl";
		xres = <800>;
		yres = <600>;
		bpp = <32>;
	};
